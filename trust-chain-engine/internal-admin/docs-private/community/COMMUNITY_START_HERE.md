# TrustChain Community - Start Here

If you are new here, read this page first.

## 1) What We Are Building

TrustChain is an open settlement protocol focused on verifiable execution, transparent incentives, and long-term network value.

Core direction:

- open protocol
- transparent governance
- utility-first tokenomics
- community-led long-term growth

1. Read protocol basics:

   - `docs/OPENCLOW_V01_DEPLOY_TEST_INSTRUCTIONS.txt`
3. Read how contribution works:

## 3) How Contribution Works

Simple flow:

1. Pick task
2. Confirm ownership in issue thread
3. Build and test
4. Submit PR with verification notes
5. Review and merge
6. Receive recognition/reward (if bounty)

Standards:

- focused scope
- clear test/verification steps
- no hidden behavior changes
