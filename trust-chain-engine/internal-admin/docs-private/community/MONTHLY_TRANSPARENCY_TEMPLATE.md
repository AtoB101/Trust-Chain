# Karma Monthly Transparency Report Template

## Report Metadata

- Month:
- Report Owner:
- Publish Date:
- Scope Version:

## 1) Product and Protocol Progress

- key milestones completed
- key releases shipped
- major blockers and resolution status

## 2) Technical Health

- settlement count
- settlement success/failure rate
- latency highlights
- incidents and postmortem links

## 3) Community and Contributors

- active contributors count
- merged PR count
- top contribution categories
- notable new contributors

## 4) Incentives and Bounties

- total bounties opened
- total bounties paid
- payout summary by level (L1/L2/L3)
- pending bounty queue

## 5) Tokenomics Snapshot

- gross emission
- burned amount
- net issuance
- treasury inflow/outflow
- staking/slashing summary (if applicable)

## 6) Governance Updates

- proposals opened
- proposals passed/rejected
- parameter changes executed
- pending timelock actions

## 7) Next-Month Plan

- top 3 priorities
- expected releases
- support needed from community

## 8) Appendix

- links to release notes
- links to dashboards/artifacts
- links to incident reports and proposal discussions
